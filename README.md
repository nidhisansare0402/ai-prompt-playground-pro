# AI Prompt Playground Pro

An AI-powered prompt playground built using Python and the Gemini API.

This project is part of my Agentic AI learning journey, where I am building production-style AI applications from scratch.

---

## Features

### Feature 1
- Connects to the Gemini API
- Generates AI responses from user prompts
- Environment variable support using `.env`

### Feature 2
- Prompt Builder
- Structured Output Support
- JSON Response Parsing
- Markdown Response Support
- Modular Project Architecture
